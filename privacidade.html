Política de privacidade

Coletamos apenas os dados necessários para operar a lista de acesso e responder contatos enviados pelo site.

Dados coletados

Nesta fase, o principal dado coletado é o e-mail informado voluntariamente pelo usuário na lista de acesso ou nos formulários de contato.

Uso dos dados

Os dados são utilizados para enviar atualizações relevantes sobre o Farpa, comunicar novidades do early access e responder solicitações legítimas.

Solicitações

Para atualizar ou remover seus dados, utilize os canais informados na página de contato.
